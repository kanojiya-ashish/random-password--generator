# Random Password Generator

A simple **Random Password Generator** that creates strong, secure passwords using a mix of characters.  
Useful for improving account security and avoiding weak passwords.

---

## Features

- Generates random passwords instantly
- Supports:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Customizable password length
- Lightweight and easy to use

---

## How It Works

The generator randomly selects characters from a predefined character set and combines them to form a secure password of the desired length.

No passwords are stored. Everything is generated on the spot.

---

## Usage

1. Run the program / open the file
2. Enter the desired password length
3. The program generates a strong random password
4. Copy and use it wherever needed
